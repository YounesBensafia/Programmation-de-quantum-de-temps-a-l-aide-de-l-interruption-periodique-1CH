# Programmation-de-quantum-de-temps-a-l-aide-de-l-interruption-periodique-1CH
C’est quatre programmes MODULAIRES dénommés “1_tache”, “1_tache2”, “5_tache”, 5_tache2"
permettant, à l’aide de l’interruption 1CH, de provoquer la suspension du programme principal en cours d’exécution
et l’exécution chaque seconde d’une procédure qui affichera:
1_tache:
  1_tache: le message ‘1 sec écoulée…’.
  1_tache2: le message ‘1 sec écoulée…’ avec 5 min d'exécution seulement
5_tache:
  5_tache: affichera cinq messages:
    Message de la tache 1 : ‘Tache 1 est en cours d’exécution....’
    Message de la tache 2 : ‘Tache 2 est en cours d’exécution....’
    Message de la tache 3 : ‘Tache 3 est en cours d’exécution....’
    Message de la tache 4 : ‘Tache 4 est en cours d’exécution....’
    Message de la tache 5 : ‘Tache 5 est en cours d’exécution....’
5_tache2: affichera les cinq messages avec 5 min d'exécution seulement
